<p align="center">Easily backup your Laravel application database!</p>


Current Support: MySQL , Disk: local, s3

## Documentation
Installation
```
composer require haruncpi/laravel-db-backup
```

Backup
```
php artisan db:backup
```
Backup to S3
```
php artisan db:backup s3
```
